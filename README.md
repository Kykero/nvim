# Ma configuration Neovim

Ceci est ma configuration personnelle de **Neovim** basée sur **NvChad**, avec quelques LSP configurés pour améliorer l'expérience de développement.

---

## Présentation

- **Neovim** : Éditeur de texte moderne et extensible.
- **NvChad** : Framework de configuration rapide et esthétique pour Neovim.
- **LSP (Language Server Protocol)** : Fournit l'autocomplétion, la navigation dans le code et les diagnostics.

---

## Installation

1. Installer Neovim (>= 0.9 recommandé) :  
   ```bash
   # Sur Linux (exemple Arch)
   sudo pacman -S neovim
   ```
2. Installer NvChad :
    ```
    git clone https://github.com/NvChad/NvChad ~/.config/nvim --depth 1 
    nvim
    ```
